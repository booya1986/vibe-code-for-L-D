# מהפכת הוייב-קודינג | The Vibe Coding Revolution

A fullscreen HTML slideshow presentation about how non-tech L&D professionals become builders in the AI era using Vibe Coding.

**[View the live presentation](https://booya1986.github.io/vibe-code-for-L-D/)**

## Quick Start

Just open `slideshow.html` in any modern browser — no build tools, no server required. Or view it online via [GitHub Pages](https://booya1986.github.io/vibe-code-for-L-D/).

## What's Inside

A 29-slide interactive presentation covering:

| Part | Topic | Slides |
|------|-------|--------|
| **Intro** | The AI revolution & context | 1–9 |
| **Part 1** | Getting into Vibe Coding — Ideation, PRD, Agentic Workflows, MVP | 10–14 |
| **Part 2** | The Builder's Toolbox — Lovable, Claude Code, Cursor, Gemini | 15–16 |
| **Part 3** | The New L&D — Content, AI Layer, Data, Agents & Automation | 17–21 |
| **Part 4** | Architecture 101 & Claude Code deep dive | 22–25 |
| **Part 5** | Builder mindset, checklist & closing | 26–29 |

## Features

- **RTL Hebrew** layout (Heebo font)
- **Dark theme** with neon green (`#22C55E`) accent and glow effects
- **Keyboard navigation** — Arrow keys (←/→), swipe on touch devices
- **Dot navigation** & progress bar
- **Sound effects** on specific slides
- **Responsive** slide transitions with CSS animations
- **Zero dependencies** — single HTML file, works offline

## Navigation

| Action | Control |
|--------|---------|
| Next slide | `←` key, swipe left, or click left arrow |
| Previous slide | `→` key, swipe right, or click right arrow |
| Jump to slide | Click any dot in the bottom nav |

> Navigation directions are intentionally swapped for RTL reading flow.
